How to use

Start the program. Click New.

Choose one of the basic figures:

  • Square
  • Square2
  • Square90
  • Triangle
  • Diamond
  • Brick
  • SquareN8 (a square and a octagon

Click OK. Select the figure. Click Edit.
Now you can start editing the figure. Click in de middle of a line.
Hold done the pen. And move the pen. Two squares appear, and move together.
Select color from the tessellate menu. The program will tessellate the figure.
Select edit from the tessellate menu. Now you can edit the figure.
To save your figure select Done from the tessellate menu.

All devices with Palm OS < 3.5 don't work.

The devices from Sony or Palm OS >= 5.0 aren't tested. But should work in a low resolution mode. (hires assist disabled)

It works on the sony clie if the hires assist is not enabled for it.

Features

  • 7 different basic figures
  • Loading, saving, removing and rename from database
  • Non-Recursive color fill
  • Removing points
  • Fit Figure, zoom in and out
  • Shortcuts
  • Beam figures
  • Icon

Bugs

  • 'Tesselation' is misspelled. The correct spelling is Tessellation.
  • Hires doesn't fill correctly

But if you find a bug, send a bug report to David Rasch.

Information to include in bug report:

  • Version of Tessellation
  • Version of Palm OS
  • Model palm (Palm IIIc, Palm V, Handspring, Tungsten Sony Clië, etc.)
  • Display capabilities (b/w, gray scale (2 or 4 bit), color (8 or 16 bit), resolution (160x160, 320x320, etc.)
  • How to reproduce the bug
