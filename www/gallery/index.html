How to create a screenshot

Download FigureDB.pdb from your palm

  • Palm Desktop : Look in the Palm desktop backup directory.
  • Linux/Unix : pilot-xfer -f FigureDB.pdb

Using pose

  1. Start pose
  2. Install the tesselation.prc and FigureDB.pdb
  3. Edit and Tessellate the figure.
  4. Right click and select Save Screen As.
  5. Convert file to a jpg.

Using pytessel

  1. Download and install pytessel on your pc. (howto see pytessel).
  2. Copy FigureDB.pdb to the pytessel directory
  3. Run pytessel.py
